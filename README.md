# create-standard

Create standard modern JavaScript/Node project easily.

## Usage

For npm users:

```
npm init standard
```

For yarn users:

```
yarn create standard
```

## Features

- `prettier` and `eslint` integration

- `commitizen` and `commitlint` integration

- commonly used dot files for modern projects
